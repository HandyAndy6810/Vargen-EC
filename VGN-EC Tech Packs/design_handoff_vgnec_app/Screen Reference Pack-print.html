Home · Mission Control

mobile/app/(tabs)/index.tsx

Data

  • HooksuseAuth(), useJobs()
  • DerivedfirstName, weekDays, upcomingJobsCount, nextJob, selectedDateJobs, getDayBadge
  • New depsnone — reuse existing logic

Layout (top → bottom)

  • Status bariOS native, dark content, time + dynamic island area transparent over warm bg
  • Greeting"G'day, {firstName}." — large 32px Manrope ExtraBold, brand-600 on the name. Subline ink-500 15px dynamic: jobs today count + weekly-goal gap
  • Weather chipRight-aligned small pill with sun icon + °C + city. Optional — hide if no geolocation.
  • UP NEXT heroDark card (bg-ink-900), warm accent glow. Eyebrow UP NEXT · {time}. Title + sublocation. Right: duration chip. Actions row: PrimaryButton "Start job" + 2 icon buttons (nav pin, chat)
  • Weekly goal barHorizontal progress — $ earned this week vs userSettings.weeklyGoal. Soft brand-500 fill on ink-100 track.
  • Activity bladesStacked cards: pipeline snapshot, quick actions, revenue this month, calendar strip. Order from userSettings.bladeOrder JSON array
  • Floating tab barBlurView, icon-only (5 tabs), active dot in brand-600. mx-4 mb-6 rounded-3xl

Critical details

  • AvatarTop-left 44px circle, initials on brand-100 bg / brand-700 text. Tap → /profile
  • Hero copyNever invent numbers. Hide weekly-goal bar if weeklyGoal === 0. Hide UP NEXT card if !nextJob — replace with empty hero.
  • Blade orderPersist re-ordering via PATCH /api/settings body { bladeOrder: "[...]" }
